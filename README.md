# ItzGalaxy

A personal project hub by **Pixl** for self-hosted tools, Discord bots, desktop applications, and game servers. Everything you need in one place.

## Project Structure

```
frontend/
├── src/
│   ├── components/     # Reusable React components
│   ├── pages/          # Page components (Home, About, Resources, etc)
│   ├── assets/         # Fonts and static assets
│   ├── App.tsx         # Main app component
│   ├── main.tsx        # Entry point
│   └── index.css       # Global styles
├── public/             # Static files (logos, backgrounds, console)
├── tailwind.config.js  # Tailwind CSS configuration
├── postcss.config.js   # PostCSS configuration
├── vite.config.ts      # Vite configuration
├── tsconfig.json       # TypeScript configuration
└── package.json        # Dependencies and scripts
```

Built with React 19, TypeScript, Vite, and Tailwind CSS 4.2.
